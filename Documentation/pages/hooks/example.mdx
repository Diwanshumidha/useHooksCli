# useDebounce Hook

The `useDebounce` hook is a utility that helps debounce a value, allowing you to delay the execution of a function until after a specified amount of time has elapsed since the last time the debounced function was invoked.

## Usage

```jsx
import { useDebounce } from 'hooks-boost';

const [val,setVal] = useState("")
const debouncedValue = useDebounce(val, 300);
```

In this example, `debouncedValue` will be the debounced version of `val`, and it will only be updated after 300 milliseconds of inactivity in the value of `val`.

## Parameters

- `value` (any): The value you want to debounce.
- `delay` (number): The time in milliseconds to wait before updating the debounced value.

## Returns

- `debouncedValue` (any): The debounced value that will be updated after the specified delay.

## Example

```jsx
import { useDebounce } from 'hooks-boost';

function DebouncedComponent() {
  const inputValue = /* your input value */;
  const debouncedInputValue = useDebounce(inputValue, 300);

  return (
    <div>
      <p>Input Value: {inputValue}</p>
      <p>Debounced Value: {debouncedInputValue}</p>
    </div>
  );
}
```

In this example, the `DebouncedComponent` displays both the raw input value and the debounced value after using the `useDebounce` hook.
